Installation

・ Installation
 Download and run the "msi" file to begin the installation process.
 Once installation is complete,
 click the Start button (Windows logo) located on the taskbar.
 You will find the application icon displayed in the Start menu.
 
・ Uninstall
 Right-click the app icon in the Start menu,
 then select Uninstall from the menu that appears.
 
 
System Requirements 
 64-bit version of Windows 10 or Windows 11
